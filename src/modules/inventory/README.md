# Inventory Module

## Overview
Track parts, supplies, and materials used on job items.

## Planned Features
- Parts catalog management
- Stock level tracking
- Parts used per job item
- Low stock alerts
- Reorder suggestions
- Cost tracking per job

## Status
**Not yet implemented**

- `hooks/useInventory.ts` - Hook for inventory operations
- `services/inventoryService.ts` - Database operations for inventory
- `components/` - UI components for inventory management
