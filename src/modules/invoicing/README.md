# Invoicing Module

## Overview
Generate invoices and track payments for completed jobs.

## Planned Features
- Invoice generation from job items
- Line items from labor + parts
- Customer billing info
- Payment tracking
- Invoice PDF export
- Payment reminders
- Integration with payment processors
